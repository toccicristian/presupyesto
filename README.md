`python3 -m PyInstaller --onedir --contents-directory . presupyesto.py`

###y en windows copiamos presupyesto.py como presupyesto.pyw y compilamos con:
`python3 -m PyInstaller --onedir --contents-directory . presupyesto.pyw`
